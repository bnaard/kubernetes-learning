# Security policy

Kubernetes Learning is an educational document, not a Kubernetes cluster,
operator, or production deployment tool. Examples may describe privileged
operations; apply them only to an explicitly disposable or authorized cluster.

Do not publish credentials, kubeconfig files, private infrastructure details,
or unpatched security findings in public issues. Report concerns privately
through the [bnaard GitHub profile](https://github.com/bnaard) with the affected
path, commit, safe reproduction steps, and suggested mitigation.

Only the latest commit on `main` is supported. There are no tagged GitHub
source releases; published PDFs are maintained separately on Codeberg.
